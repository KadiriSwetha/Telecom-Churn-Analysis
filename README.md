# Telecom-Churn-Analysis

Project Overview
The Telecom Churn Analysis project focuses on analyzing customer behavior and identifying the factors that contribute to customer churn in the telecommunications industry. Customer churn refers to customers leaving or discontinuing telecom services. Understanding churn patterns helps telecom companies improve customer retention and reduce business losses.

This project uses SQL for data analysis and Microsoft Excel for dashboard creation and visualization. The dataset contains over 243,000 customer records, including customer demographics, telecom partner details, service usage information, salary details, and churn status.

The analysis includes customer segmentation, churn rate analysis, usage behavior analysis, and state-wise performance evaluation. Different SQL queries were used to extract meaningful insights such as churn percentage by telecom provider, customer usage trends, gender-wise churn distribution, and high-risk customer groups.

An interactive Excel dashboard was created to visualize important KPIs like total customers, churn rate, average data usage, calls made, and customer distribution across different telecom partners and states. The project helps demonstrate practical skills in SQL querying, data cleaning, business analysis, and dashboard reporting using Excel.
